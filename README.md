# AT STUDIO Launch Page

Landing page for AT Studio launch - April 1st, 2026

## Features

- 3D particle animation morphing to logo (Desktop only)
- Animated starfield background
- Responsive design (optimized for mobile/tablet)
- Pilot program application form
- Countdown timer

## Tech Stack

- Astro
- React
- Three.js
- GSAP
- TailwindCSS
- Lenis (smooth scroll)

## Development

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm run preview
```
